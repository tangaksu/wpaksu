== 简介 ==
aksu defense 是一款为 WordPress 量身打造的轻量级网站安全防护插件，集成 CC 攻击防御、SQL/XSS 注入拦截、IP 黑白名单、敏感路径拦截、恶意文件上传检测等核心防护功能。

== 主要功能 ==
- CC 攻击防护和速率限制
- SQL/XSS 注入攻击拦截
- 恶意 User-Agent 拦截与UA黑名单自定义
- 敏感路径和扫描行为拦截
- Cookie 注入防护
- 文件上传扩展名和内容检测
- IP 黑白名单支持（支持通配符）
- 实时安全日志记录与导出
- 文件变化监控（快照/差异）
- 后台仪表盘统计与管理

== 安装 ==
1. 上传插件文件夹到 `/wp-content/plugins/` 目录。
2. 后台启用“aksu defense”插件。
3. 在“安全防御”菜单下进行安全策略配置。

== 常见问题 ==
= 插件会影响站点性能吗？ =
插件为轻量级设计，对性能影响极低。各项防护均可按需开启/关闭。

= 如何排查误拦截？ =
请在“安全日志”页面查看详细记录，根据类型/请求调整白名单或防护策略。
